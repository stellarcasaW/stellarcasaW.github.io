submitXDR

Application to submit Signed XDR manually in simple Command Line interface

Features

Lightweight application for submitting Signed Stellar Transactions.

  • Supports submitting any XDR
  • Supports Horizon Public and Test Network
  • Prints Response and Ledger Information for Success Transactions

Dependencies

  • Stellar Go SDK

Usage

  1. Clone the repo
git clone github.com/aanupam23/submitXDR
  1. Run this application using
go run submitXDR.go
